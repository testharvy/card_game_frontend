# card_game_frontend


### Фронтенд "коллекционная карточная игра"
Простая игра без цели.
Каждый игрок имеет свою колоду карт, которые он может покупать, продавать и объединять в более прокаченную версию.





- react
- typescript
- react-dom-router
- redux
- vite
